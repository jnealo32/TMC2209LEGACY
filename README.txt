TMC2209LEGACY
VERSION 1.0.0
Author: Joseph Neal
Date: 2025-03-21


Description:
-----------
Simple Arduino library for driving the TMC2209 operated in Legacy Mode.


Notes:
-----------
Documentation can be found in the header file in "src/"

Be sure to place entire TMC2209LEGACY folder inside of Arduino/libraries

Alternatively, folder can be zipped and manually included by going to:
Sketch -> Include Library -> Add .ZIP Library...

A basic example is provided for controlling 2 motors simultaneously
